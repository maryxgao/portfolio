I'm Mary Gao. I study Computer Science and Philosophy at Harvard (2020). I'm here to make the world better (cheesy but true).

  • Web-based Facial Authentication

    The goal of the project was to create a proof-of-concept for website-based facial authentication. This demo is of a mock HarvardKey login page. Using the computer's webcam, a photo is taken and matched against the Harvard students database (school year 2018-2019). If a match is found, the user "logs into" my.harvard.edu. A user can also register a new photo if they are not already in the database.

    Github: private - to protect student privacy

    Date: November 2018 - December 2018

    Team members: Matthew Finlayson, Winston Michalak

    Languages: JavaSript, Python, HTML, CSS

  • 5G Access Point Optimization at Harvard

    Motivated by using existing infrastructure from Wi-Fi routers to reduce costs of the eventual installation of 5G access points, fixed placement of 5G APs is contrasted with dynamic, algorithmic placement. The goal was to find optimal 5G AP placement that maximizes the evaluation metrics being median power and median throughput. The approach was to run simulations based on a Harvard dining hall to determine optimal performance given varying scenarios.

    Date: November 2018 - December 2018

    Team members: Thomas Culp

    Languages: Python

    Date: December 2018 - January 2019

  • Travel Inspiration Chrome Extension

    A chrome extension such that when a new tab is opened, an 3x3 grid of HD photos of a city along with the cheapest flight to that place from where you are currently located. The cities are chosen from a pre-determined array based on cities that I would like to travel to.

    Date: First 1.5 weeks of January 2019

    Languages: JavaScript, HTML, CSS

    API's: Skyscanner, Unsplash, Lufthansa, Google Geocode
